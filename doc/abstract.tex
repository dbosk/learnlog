The \texttt{learnlog} package provides automatic logging of student code
development and test runs.
By adding \mintinline{python}{import learnlog} as the first import in their
Python files, students automatically record every program run: source code
changes, command-line arguments, standard input/output/error, and unhandled
exceptions.
The data is stored in a hidden local Git repository, giving a complete
timeline of how students develop and debug their code.
Logs can be shared via Git remotes: a teacher can push a live-coding
session for students to clone and replay, and a researcher can collect
student logs by having each student push to a designated repository.
